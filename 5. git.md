# Git
## 常用指令
### 配置相关
> 最小化配置（全局配置）
```
git config --global user.name "snoredude"
git config --global user.email "caffeine.dude.go@gmail.com"
```

> 最小化配置（当前 git 仓库）
```
git config --local user.name "snoredude"
git config --local user.email "caffeine.dude.go@gmail.com"
```

> 查看配置信息
```
git config --global --list
git config --local --list
```

> 代理
```
git config --global http.proxy 127.0.0.1:8087
git config --global http.sslVerify false
git config --global --unset http.proxy
git config --global --unset http.sslVerify
```

> 配置文件
```
git config --global --edit
```

# Git

## 常用命令
查看当前代码所在分支：`git branch -vv`  
列出所有本地分支：`git branch -v`  
列出所有远端分支：`git branch -rv`  
列出所有本地分支 + 远端分支：`git branch -av`
